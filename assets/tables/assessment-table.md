# 结构化评估打分表

**说明**:此表用于对照评分标准，对伴侣格局进行多维度打分。建议结合具体案例使用。

---

## 评分指南

### 评分标准
- **1-3分**: 极差/严重问题
- **3-5分**: 较差/明显问题
- **5-7分**: 一般/基本合格
- **7-9分**: 良好/优秀
- **9-10分**: 极好/卓越

### 打分原则
- 基于具体案例，避免主观臆断
- 至少需要3-6个月的行为样本
- 用客观事实支撑打分
- 可以打小数（如6.5分）

## 类型判定标准

### 丫鬟型特征
- 情绪稳定性: ≤3分
- 成长支持度: ≤3分
- 价值独立性: ≤3分
- 典型得分区间: 2.0-4.0分

### 妃子型特征
- 自我价值锚点: ≤3分
- 价值独立性: ≤5分
- 情绪稳定性: ≥5分（表面稳定）
- 典型得分区间: 3.0-5.0分

### 皇后型特征
- 情绪稳定性: ≥7分
- 成长支持度: ≥7分
- 价值独立性: ≥7分
- 自我价值锚点: ≥7分
- 典型得分区间: 6.5-9.5分

## 风险等级判定

| 得分区间 | 风险等级 | 描述 | 建议 |
|---------|---------|------|------|
| ≤4分 | 高风险 | 严重拖垮关系 | 建议立即止损或深度干预 |
| 4-6分 | 中风险 | 有消耗但可能调整 | 建议观察+沟通+边界设定 |
| 6-8分 | 低风险 | 基本健康 | 建议持续优化 |
| ≥8分 | 优质 | 皇后型伴侣 | 建议珍惜并深度合作 |

## 使用示例

### 示例:丫鬟型伴侣打分

#### 维度打分
- **情绪稳定性**: 2分（频繁抱怨，情绪爆发）
- **成长支持度**: 2分（阻拦创业，要求稳定）
- **价值独立性**: 2分（无独立事业，依附我）
- **自我价值锚点**: 3分（通过照顾家庭获得价值感）
- **危机应对能力**: 3分（抱怨但不离开）
- **长期匹配度**: 2分（价值观差异大）

#### 综合得分计算
```
综合得分 = 2×0.25 + 2×0.20 + 2×0.15 + 3×0.15 + 3×0.10 + 2×0.15
        = 0.5 + 0.4 + 0.3 + 0.45 + 0.3 + 0.3
        = 2.25分
```

#### 判定结果
- **类型**: 丫鬟型（低情绪稳定性+低成长支持度+低价值独立性）
- **风险等级**: 高风险
- **建议**: 结合关系时长和野心程度评估止损时机
