# 行为信号对照表

**说明**:此表用于快速对照伴侣行为信号，识别格局类型。建议结合具体案例使用。

---

## 丫鬟型行为信号对照

### 信号1:抱怨陪伴时间

| 特征 | 表现 | 对比测试 |
|------|------|---------|
| **识别要点** | 事业上升期，抱怨陪她时间少了 | 加班深夜，第一反应是什么？ |
| **丫鬟型反应** | "你心里还有没有这个家" | 抱怨、情绪勒索 |
| **皇后型反应** | "你辛苦了" | 关心、理解 |
| **判断标准** | 是在关心你的状态，还是在限制你的发展？ | |

**观察要点**:
- 你加班时，她的第一反应
- 你忙于工作时，她的态度
- 你取得成就时，她是否替你高兴

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 我连续加班一周
- 伴侣反应: "你心里还有没有这个家"
- 信号类型: 丫鬟型信号1
- 得分: 2分

---

### 信号2:阻拦风险决策

| 特征 | 表现 | 典型话语 |
|------|------|---------|
| **识别要点** | 阻拦有风险但有前景的决定 | 本能反应是阻拦而非一起评估 |
| **丫鬟型反应** | 阻拦、恐惧、要求稳定 | "别折腾了，稳定点多好" |
| **皇后型反应** | 支持、评估、共同承担 | "我们一起评估一下风险" |
| **判断标准** | 是否一起评估风险和收益？ | |

**观察要点**:
- 你想创业时，她的反应
- 你想做风险投资时，她的态度
- 你要跳槽时，她的意见

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 我想辞职创业
- 伴侣反应: "别折腾了，稳定点多好，万一失败了怎么办"
- 信号类型: 丫鬟型信号2
- 得分: 2分

---

### 信号3:控制欲增强

| 特征 | 表现 | 核心机制 |
|------|------|---------|
| **识别要点** | 你变好时，她越来越焦虑、想控制 | 格局承载不了一个不断进化的你 |
| **丫鬟型反应** | 焦虑、控制、要求妥协 | "你别那么拼了" |
| **皇后型反应** | 支持、引导、共同成长 | "我支持你" |
| **判断标准** | 她是否一起成长，还是控制你？ | |

**观察要点**:
- 你越来越强时，她的变化
- 你社交时，她的态度
- 你与他人合作时，她的反应

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 我事业越来越成功
- 伴侣反应: 越来越焦虑，要求我少加班，控制我的社交
- 信号类型: 丫鬟型信号3
- 得分: 2分

## 妃子型行为信号对照

### 信号1:完美展示无真实

| 特征 | 表现 | 对比测试 |
|------|------|---------|
| **识别要点** | 永远展示精致完美状态 | 在一起很久，你了解真实她吗？ |
| **妃子型反应** | 精致展示，无真实 | 说不清她在意什么、害怕什么 |
| **皇后型反应** | 自我要求，有真实 | 了解她的内心世界 |
| **判断标准** | 她展示的是产品还是真实的人？ | |

**观察要点**:
- 她在你面前是否总是完美
- 你是否了解她真实的样子
- 她的缺点是什么

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 在一起半年了
- 伴侣反应: 永远精致完美，但我说不清她真实在意什么
- 信号类型: 妃子型信号1
- 得分: 2分

---

### 信号2:关注利益交换

| 特征 | 表现 | 风险测试 |
|------|------|---------|
| **识别要点** | 关注你能提供什么 | 事业好时vs遇到波折时 |
| **妃子型反应** | 关注消费、资源、生活品质 | 事业好时态度好，波折时态度变 |
| **皇后型反应** | 关注人本身、共同成长 | 态度稳定 |
| **判断标准** | 她关注的是你这个人还是你的资源？ | |

**观察要点**:
- 她对你的关注度变化
- 你有钱时vs没钱时，她的态度
- 你有权时vs无权时，她的反应

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 我创业遇到资金困难
- 伴侣反应: 态度立即转变，不如以前热情
- 信号类型: 妃子型信号2
- 得分: 2分

---

### 信号3:制造危机感

| 特征 | 表现 | 潜台词 |
|------|------|-------|
| **识别要点** | 有意无意提其他追求者 | 让你知道她不缺选择 |
| **妃子型反应** | "今天又有人约我了" | 制造竞争焦虑 |
| **皇后型反应** | 分享生活，制造危机感 | 无此行为 |
| **判断标准** | 她是在分享还是在制造危机感？ | |

**观察要点**:
- 她是否经常提其他男性
- 提的目的是什么
- 是否让你产生竞争焦虑

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 日常对话
- 伴侣反应: "今天又有人约我了，前男友以前对我可好了"
- 信号类型: 妃子型信号3
- 得分: 2分

## 皇后型行为信号对照

### 信号1:共情解决问题

| 特征 | 表现 | 典型话语 |
|------|------|---------|
| **识别要点** | 遇到困难时一起想办法 | 第一反应不是慌/抱怨/甩锅 |
| **皇后型反应** | 共情、分析、解决 | "现在情况是什么，我们怎么处理" |
| **丫鬟型反应** | 抱怨、情绪勒索 | "你怎么搞成这样" |
| **判断标准** | 她是解决问题还是制造问题？ | |

**观察要点**:
- 你遇到困难时，她的第一反应
- 她是否能提供实际帮助
- 她是否能稳定你的情绪

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 我项目失败
- 伴侣反应: 立即一起分析原因，想办法解决
- 信号类型: 皇后型信号1
- 得分: 8分

---

### 信号2:独立价值判断

| 特征 | 表现 | 核心能力 |
|------|------|---------|
| **识别要点** | 有独立的价值和判断力 | 不因你说什么就信什么 |
| **皇后型反应** | 独立思考、有质量反馈 | 能看到你看不到的盲区 |
| **丫鬟型/妃子型反应** | 依附他人、无独立判断 | 易受影响 |
| **判断标准** | 她能否提供有质量的反馈？ | |

**观察要点**:
- 她是否有独立意见
- 她的判断是否有价值
- 她是否能发现盲区

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 讨论我的创业方案
- 伴侣反应: 提出独立见解，发现我没想到的盲区
- 信号类型: 皇后型信号2
- 得分: 8分

---

### 信号3:校准方向能力

| 特征 | 表现 | 核心定位 |
|------|------|---------|
| **识别要点** | 最膨胀时敢泼冷水，最低谷时能兜底 | 不是围着你情绪转，是帮你校准方向 |
| **皇后型反应** | 泼冷水、兜底、校准方向 | 客观、理性 |
| **丫鬟型/妃子型反应** | 情绪化、无底线 | 跟随情绪 |
| **判断标准** | 她是帮你校准方向还是跟随情绪？ | |

**观察要点**:
- 你膨胀时，她的反应
- 你低谷时，她的反应
- 她是否能客观评价你

**记录示例**:
- 日期: YYYY-MM-DD
- 事件: 我取得重大成功，有些膨胀
- 伴侣反应: 敢泼冷水，提醒我保持清醒
- 信号类型: 皇后型信号3
- 得分: 8分

## 误判点对照表

### 误判1:关心身体 vs 限制发展

| 类型 | 表现 | 关键区分 |
|------|------|---------|
| **皇后型** | "需要调整节奏，这样才能走更远" | 帮优化 |
| **丫鬟型** | "你别搞了，够了" | 消除自身恐惧 |
| **判断标准** | 是在关心你的状态，还是在限制你的发展？ | |

---

### 误判2:精致形象 vs 媚男风格

| 类型 | 精致目的 | 穿衣风格 |
|------|---------|---------|
| **皇后型** | 自我要求的一部分 | 穿自己想穿的 |
| **妃子型** | 为了卖一个好价钱 | 极致媚男（低胸短裙、突出身材曲线） |
| **验证方法** | 看她在没有观众时是否同样讲究 | 日常穿搭有没有非取悦不可的痕迹 |

---

### 误判3:阶段性表现 vs 真实格局

| 阶段 | 可能表现 | 真实格局 |
|------|---------|---------|
| **伪装期**（<6个月） | 可能展现皇后型特征 | 还在争取你 |
| **稳定期**（≥6个月） | 真实格局暴露 | 关系失去新鲜感 |
| **验证方法** | 不是看3个月，而是看关系失去新鲜感之后 | |

## 使用指南

### 快速对照流程
1. 回顾最近3个月的行为样本
2. 对照三类行为信号清单
3. 识别出现的信号类型
4. 统计各类信号数量
5. 判定主类型

### 记录要点
- 具体事件和日期
- 伴侣的具体反应
- 对应的信号类型
- 支持该信号的得分为

### 注意事项
- 避免单点判断，需要多个信号支持
- 注意误判点，使用区分标准
- 考虑阶段性表现，观察长期趋势
- 结合具体案例，避免主观臆断

---

**注意**:此表用于快速对照识别，但需要结合结构化评估模型和动态跟踪框架进行综合判断。
