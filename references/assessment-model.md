## 概览

本评估模型将伴侣格局从多个维度进行量化评估，提供客观的可对比分析工具。

**核心原则**:
- 避免单点判断，多维度综合评估
- 区分信号权重，识别关键行为模式
- 提供可量化的对比基准

**适用场景**:
- 初期关系判断（新伴侣识别）
- 现有关系评估（长期伴侣分析）
- 多伴侣对比（选择决策参考）

## 评估维度

### 维度1:情绪稳定性（权重25%）
- 定义:在压力、冲突、变化场景下的情绪反应模式
- 评估要点:
  - 遇到困难时的第一反应（慌乱/抱怨/解决）
  - 情绪波动频率与恢复速度
  - 对伴侣情绪的影响力（稳定/拉低/激化）
- 评分标准:
  - 1分:情绪化严重，频繁爆发，恢复慢
  - 3分:偶尔波动，能基本控制
  - 5分:情绪稳定，善于调节，能稳定伴侣
  - 7分:高度情绪稳定，危机时保持冷静
  - 10分:情绪教练级别，能主动调节系统情绪

### 维度2:成长支持度（权重20%）
- 定义:对伴侣成长、事业发展的支持方式与程度
- 评估要点:
  - 对伴侣事业的反应（支持/阻拦/控制）
  - 是否提供资源或人脉支持
  - 是否与伴侣共同成长
- 评分标准:
  - 1分:阻拦、限制、抱怨不陪伴
  - 3分:不反对但无支持，要求稳定
  - 5分:表面支持，但影响有限
  - 7分:积极支持，提供实际帮助
  - 10分:战略伙伴，能调度资源共同成长

### 维度3:价值独立性（权重15%）
- 定义:是否具备独立的价值体系和判断能力
- 评估要点:
  - 是否有独立兴趣爱好/事业/社交圈
  - 是否因外界评价轻易动摇
  - 能否提供有质量的反馈和建议
- 评分标准:
  - 1分:完全依附，无独立判断
  - 3分:有部分独立性，但易受影响
  - 5分:有基本独立思考，但深度有限
  - 7分:独立性强，能提供有质量反馈
  - 10分:高度独立，能发现盲区，引领思考

### 维度4:自我价值锚点（权重15%）
- 定义:自我价值来源（外部认可 vs 内在标准）
- 评估要点:
  - 形象管理的目的（自我要求 vs 取悦他人）
  - 对外界评价的敏感度
  - 在无观众状态下的行为一致性
- 评分标准:
  - 1分:完全依赖外部认可，表演型人格
  - 3分:在意评价，但有部分自我标准
  - 5分:内外平衡，但仍有较强表现欲
  - 7分:以内在标准为主，适度社交要求
  - 10分:完全内在驱动，行为高度一致

### 维度5:危机应对能力（权重10%）
- 定义:在关系危机、伴侣低谷时的应对方式
- 评估要点:
  - 伴侣低谷时的反应（离开/抱怨/兜底）
  - 是否有能力共同解决问题
  - 是否能在危机中保持关系价值
- 评分标准:
  - 1分:逃避、指责、落井下石
  - 3分:抱怨但不离开，无力解决
  - 5分:愿意陪伴但无法实质帮助
  - 7分:能提供情绪支持和实际帮助
  - 10分:战略伙伴，能共同破局

### 维度6:长期匹配度（权重15%）
- 定义:与伴侣长期发展目标的契合程度
- 评估要点:
  - 人生规划一致性
  - 价值观契合度
  - 对未来的共同愿景
- 评分标准:
  - 1分:完全不匹配，未来冲突严重
  - 3分:部分匹配，但存在重大分歧
  - 5分:基本匹配，能协商调整
  - 7分:高度匹配，能共同规划
  - 10分:完全契合，人生合伙人

## 计算方法

### 综合得分计算
```
综合得分 = Σ(维度得分 × 权重)
= 情绪稳定性×0.25 + 成长支持度×0.20 + 价值独立性×0.15 + 自我价值锚点×0.15 + 危机应对能力×0.10 + 长期匹配度×0.15
```

### 类型判定标准

**丫鬟型特征**:
- 情绪稳定性: ≤3分
- 成长支持度: ≤3分
- 价值独立性: ≤3分
- 典型得分区间: 2.0-4.0分

**妃子型特征**:
- 自我价值锚点: ≤3分
- 价值独立性: ≤5分
- 情绪稳定性: ≥5分（表面稳定）
- 典型得分区间: 3.0-5.0分

**皇后型特征**:
- 情绪稳定性: ≥7分
- 成长支持度: ≥7分
- 价值独立性: ≥7分
- 自我价值锚点: ≥7分
- 典型得分区间: 6.5-9.5分

### 风险等级判定

**高风险（≤4分）**:
- 严重拖垮关系
- 建议立即止损或深度干预

**中风险（4-6分）**:
- 有消耗，但存在调整空间
- 建议观察+沟通+边界设定

**低风险（6-8分）**:
- 基本健康，有成长潜力
- 建议持续优化

**优质（≥8分）**:
- 皇后型伴侣
- 建议珍惜并深度合作

## 使用流程

### 步骤1:准备数据
- 收集至少3-6个月的行为样本
- 覆盖不同场景（压力、成功、失败、冲突）
- 记录具体案例和对话

### 步骤2:维度评分
- 对每个维度打分（1-10分）
- 依据评估要点，参考评分标准
- 用具体案例支撑打分

### 步骤3:计算综合得分
- 按权重计算综合得分
- 判定类型特征
- 确定风险等级

### 步骤4:分析短板
- 识别得分最低的维度
- 分析根本原因（能力问题 vs 意愿问题 vs 格局问题）
- 评估改进可能性

### 步骤5:决策参考
- 结合关系时长、投入成本、伴侣成长意愿
- 参考决策支持矩阵（见decision-matrix.md）
- 制定行动计划

## 示例

### 示例1:丫鬟型伴侣评估

**数据收集**（6个月观察）:
- 情绪稳定性:用户加班时抱怨"你心里还有没有这个家"，频繁情绪爆发 → 2分
- 成长支持度:用户想创业时阻拦说"稳定点多好" → 2分
- 价值独立性:无独立事业，依附用户 → 2分
- 自我价值锚点:通过照顾家庭获得价值感，但在用户成长时焦虑 → 3分
- 危机应对能力:用户低谷时抱怨但不离开 → 3分
- 长期匹配度:价值观差异大，未来规划冲突 → 2分

**计算综合得分**:
= 2×0.25 + 2×0.20 + 2×0.15 + 3×0.15 + 3×0.10 + 2×0.15
= 0.5 + 0.4 + 0.3 + 0.45 + 0.3 + 0.3
= **2.25分**

**判定结果**:
- 类型:丫鬟型（低情绪稳定性+低成长支持度+低价值独立性）
- 风险等级:高风险
- 建议:结合关系时长和用户野心程度评估止损时机

### 示例2:皇后型伴侣评估

**数据收集**（1年观察）:
- 情绪稳定性:用户遇到困难时一起想办法，情绪稳定 → 8分
- 成长支持度:支持用户事业，能调度资源 → 8分
- 价值独立性:有独立事业，能提供有质量反馈 → 8分
- 自我价值锚点:内在标准驱动，行为一致 → 9分
- 危机应对能力:用户低谷时兜底，共同破局 → 9分
- 长期匹配度:人生规划高度契合 → 9分

**计算综合得分**:
= 8×0.25 + 8×0.20 + 8×0.15 + 9×0.15 + 9×0.10 + 9×0.15
= 2.0 + 1.6 + 1.2 + 1.35 + 0.9 + 1.35
= **8.4分**

**判定结果**:
- 类型:皇后型（高情绪稳定性+高成长支持度+高价值独立性+高自我价值锚点）
- 风险等级:优质
- 建议:珍惜合作关系，深化战略伙伴关系

### 示例3:混合特征伴侣评估

**数据收集**（8个月观察）:
- 情绪稳定性:平时稳定，但压力下会爆发 → 5分
- 成长支持度:表面支持但无实际帮助 → 4分
- 价值独立性:有一定独立性，但易受外界影响 → 5分
- 自我价值锚点:在意评价，但有部分自我标准 → 4分
- 危机应对能力:愿意陪伴但无法实质帮助 → 5分
- 长期匹配度:部分匹配，存在分歧 → 4分

**计算综合得分**:
= 5×0.25 + 4×0.20 + 5×0.15 + 4×0.15 + 5×0.10 + 4×0.15
= 1.25 + 0.8 + 0.75 + 0.6 + 0.5 + 0.6
= **4.5分**

**判定结果**:
- 类型:混合型（丫鬟型基础+妃子型表象）
- 风险等级:中风险
- 建议:使用混合特征识别器进一步分析，观察6个月评估改进可能性

## 注意事项
- 评分需基于具体案例，避免主观臆断
- 至少需要3-6个月的行为样本，避免短期伪装影响
- 综合得分仅供参考，需结合具体场景和关系阶段
- 类型判定不是绝对标签，需关注动态变化
- 风险等级需要与决策支持矩阵结合使用
