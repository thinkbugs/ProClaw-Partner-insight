## 概览

本决策支持矩阵提供多维度、场景化的决策参考，帮助用户在复杂情况下做出最优关系决策。

**核心原则**:
- 决策基于数据，而非情绪
- 综合多个维度，避免单点判断
- 提供可操作的策略，而非模糊建议

**适用场景**:
- 决定是否止损
- 决定是否继续投入
- 在多个伴侣中选择
- 制定关系改进计划

## 决策维度

### 维度1:评估得分（权重30%）
**数据来源**:结构化评估模型综合得分

**得分区间**:
- ≤4分:高风险，严重拖垮
- 4-6分:中风险，有消耗但可能调整
- 6-8分:低风险，基本健康
- ≥8分:优质，成就关系

**决策权重**:30%（核心维度）

### 维度2:关系时长（权重20%）
**时长区间**:
- <6个月:伪装期，判断风险高
- 6个月-2年:基本稳定，判断相对可靠
- 2-5年:长期关系，有情感依赖
- >5年:长期关系，沉没成本高

**决策逻辑**:
- 时长越短，越容易止损
- 时长越长，沉没成本越高，但问题也更严重

**决策权重**:20%（影响止损难度）

### 维度3:改进可能性（权重20%）
**评估要素**:
- 改进意愿:是否愿意沟通、反思、改变
- 改进能力:是否有成长空间和学习能力
- 改进环境:是否支持改变（你的引导、边界设定）
- 格局限制:是否存在深层格局限制（价值观、安全感）

**改进可能性评级**:
- 高:改进意愿+能力+环境都有，格局限制弱
- 中:前三项有2项，格局限制中等
- 低:改进意愿弱，或格局限制强
- 无:改进意愿无，或格局限制极强

**决策权重**:20%（决定是否值得投入时间改进）

### 维度4:沉没成本（权重15%）
**成本要素**:
- 情感投入:时间、精力、情感
- 物质投入:金钱、资源
- 社会投入:家人朋友认可度、社交圈
- 机会成本:错过其他可能的机会

**成本评级**:
- 低:投入少，容易退出
- 中:有投入，但可以承受
- 高:投入大，退出困难
- 极高:投入极大，退出几乎不可能

**决策权重**:15%（影响止损难度和决心）

### 维度5:风险等级（权重15%）
**数据来源**:风险预警系统

**风险等级**:
- 极高（红色）:出现红色信号，评估得分≤4分
- 高（橙色）:出现2-3个橙色信号，评估得分4-5分
- 中（黄色）:出现2-4个黄色信号，评估得分5-6分
- 低（绿色）:无危险信号，评估得分≥6分

**决策权重**:15%（决定紧急程度）

### 决策类型定义

#### 类型1:立即止损
**适用场景**:
- 评估得分≤4分
- 出现红色危险信号
- 改进可能性低/无
- 关系时长短（<6个月）
- 沉没成本低

**执行策略**:
1. 明确表达关系结束
2. 拒绝任何情感勒索
3. 清理共同财产
4. 切断所有联系
5. 必要时寻求专业帮助

#### 类型2:观察后止损
**适用场景**:
- 评估得分4-6分
- 改进可能性中等
- 需要验证是否真正愿意改变
- 沉没成本不高

**执行策略**:
1. 深度沟通，明确边界和期望
2. 设定3个月观察期
3. 使用动态跟踪框架记录
4. 观察期结束重新评估
5. 无明显改善则止损

#### 类型3:积极改进
**适用场景**:
- 评估得分4-8分
- 改进可能性高
- 伴侣有意愿和能力改变
- 关系有发展潜力

**执行策略**:
1. 深度沟通，识别问题根源
2. 明确改进目标和边界
3. 提供引导和支持
4. 使用动态跟踪框架记录
5. 定期评估改进效果

#### 类型4:长期优化
**适用场景**:
- 评估得分6-8分
- 关系基本健康
- 但存在一些可优化的地方
- 沉没成本高，关系稳定

**执行策略**:
1. 识别优化空间
2. 持续沟通和调整
3. 设定长期目标
4. 深化合作关系
5. 接受不完美

#### 类型5:珍惜关系
**适用场景**:
- 评估得分≥8分
- 关系优质
- 伴侣是皇后型或高配组合

**执行策略**:
1. 珍惜当前关系
2. 深化战略伙伴关系
3. 共同规划未来
4. 扩大合作范围
5. 共同成长

## 决策流程

### 步骤1:数据收集
- 使用结构化评估模型获取评估得分
- 使用风险预警系统获取风险等级
- 记录关系时长
- 评估沉没成本
- 评估改进可能性

### 步骤2:维度评估
- 对照决策矩阵，找到对应行
- 读取决策建议类型

### 步骤3:策略制定
- 根据决策建议类型，制定具体执行策略
- 明确行动步骤、时间节点、评估标准

### 步骤4:执行与跟踪
- 执行决策策略
- 使用动态跟踪框架记录
- 定期重新评估

### 步骤5:调整与优化
- 根据执行效果调整策略
- 必要时重新进入决策流程

## 场景化策略

### 场景1:新关系（<6个月）
**特点**:
- 伪装期风险高
- 沉没成本低
- 判断不确定性大

**决策策略**:
- 评估得分≤4分 → 立即止损（无需犹豫）
- 评估得分4-6分 → 观察到6个月，使用伪装验证方法
- 评估得分6-8分 → 继续观察，使用动态跟踪框架
- 评估得分≥8分 → 继续发展，但要警惕伪装

### 场景2:中期关系（6个月-2年）
**特点**:
- 伪装期基本结束
- 评估相对可靠
- 沉没成本中等

**决策策略**:
- 评估得分≤4分 → 立即止损（除非改进可能性高+沉没成本高）
- 评估得分4-6分 → 观察后止损或积极改进（根据改进可能性）
- 评估得分6-8分 → 长期优化，深化合作
- 评估得分≥8分 → 珍惜关系

### 场景3:长期关系（2-5年）
**特点**:
- 关系稳定
- 沉没成本高
- 改进难度大

**决策策略**:
- 评估得分≤4分 → 谨慎止损（需要评估可行性）
- 评估得分4-6分 → 长期优化或积极改进（根据改进可能性）
- 评估得分6-8分 → 长期优化，接受不完美
- 评估得分≥8分 → 珍惜关系

### 场景4:超长期关系（>5年）
**特点**:
- 沉没成本极高
- 情感依赖强
- 改进可能性低

**决策策略**:
- 评估得分≤4分 → 评估可行性，谨慎决策（可能不现实）
- 评估得分4-6分 → 维持现状，降低期望
- 评估得分6-8分 → 珍惜关系，接受不完美
- 评估得分≥8分 → 珍惜关系

### 场景5:发现伪装（任何时长）
**特点**:
- 关系突然变化
- 真实格局暴露
- 决策紧急

**决策策略**:
- 伪装后得分≤4分 → 立即止损
- 伪装后得分4-6分 → 观察后止损
- 伪装后得分6-8分 → 重新评估，谨慎决策

## 示例

### 示例1:新关系立即止损

**数据收集**:
- 评估得分:3.8分
- 关系时长:4个月
- 改进可能性:低（无意愿）
- 沉没成本:低
- 风险等级:极高（红色信号1个:极端控制行为）

**决策矩阵匹配**:
- 评估得分≤4分
- 关系时长<6个月
- 改进可能性低
- 沉没成本低
- 风险等级极高

**决策建议**:立即止损

**执行策略**:
1. 明确表达关系结束（避免纠缠）
2. 拒绝情感勒索（"你要离开我就死给你看"）
3. 切断所有联系
4. 保护个人隐私和财产安全

### 示例2:中期关系观察后止损

**数据收集**:
- 评估得分:4.5分
- 关系时长:1.5年
- 改进可能性:中（有意愿，但能力有限）
- 沉没成本:中
- 风险等级:高（橙色信号2个）

**决策矩阵匹配**:
- 评估得分4-6分
- 关系时长6个月-2年
- 改进可能性中
- 沉没成本中
- 风险等级高

**决策建议**:观察后止损

**执行策略**:
1. 深度沟通，明确边界和期望
2. 设定3个月观察期
3. 使用动态跟踪框架记录
4. 观察期结束重新评估:
   - 得分提升到6分以上 → 继续关系
   - 得分无明显变化 → 止损
   - 得分下降 → 立即止损

### 示例3:长期关系积极改进

**数据收集**:
- 评估得分:5.5分
- 关系时长:3年
- 改进可能性:高（有意愿、有能力、有环境）
- 沉没成本:高
- 风险等级:中（黄色信号3个）

**决策矩阵匹配**:
- 评估得分4-6分
- 关系时长2-5年
- 改进可能性高
- 沉没成本高
- 风险等级中

**决策建议**:积极改进

**执行策略**:
1. 深度沟通，识别问题根源
2. 明确改进目标:
   - 情绪稳定性从4分提升到6分
   - 成长支持度从5分提升到6分
3. 设定边界:
   - 不阻拦事业发展
   - 不用情绪勒索
4. 提供引导和支持
5. 使用动态跟踪框架记录
6. 每3个月重新评估

### 示例4:中期关系珍惜

**数据收集**:
- 评估得分:8.2分
- 关系时长:1年
- 改进可能性:N/A（已经优秀）
- 沉没成本:中
- 风险等级:低（无危险信号）

**决策矩阵匹配**:
- 评估得分≥8分
- 关系时长6个月-2年
- 风险等级低

**决策建议**:珍惜关系

**执行策略**:
1. 珍惜当前关系
2. 深化战略伙伴关系
3. 共同规划未来
4. 扩大合作范围（事业、生活、社交）
5. 共同成长

### 示例5:超长期关系维持现状

**数据收集**:
- 评估得分:4.8分
- 关系时长:8年
- 改进可能性:低（格局限制强）
- 沉没成本:极高
- 风险等级:中（黄色信号4个）

**决策矩阵匹配**:
- 评估得分4-6分
- 关系时长>5年
- 改进可能性低
- 沉没成本极高
- 风险等级中

**决策建议**:维持现状，降低期望

**执行策略**:
1. 接受不完美
2. 降低对她的期望
3. 维持关系，但不过度投入
4. 寻求其他渠道的情感和价值支持
5. 不要尝试根本性改变（不现实）

## 注意事项
- 决策要基于数据，而非情绪
- 沉没成本不是决策的唯一因素
- 改进可能性评估要客观，不要一厢情愿
- 立即止损要果断，不要拖延
- 观察期要严格执行，不要无限延长
- 长期关系要接受不完美，不要追求完美
- 最终决策权在你，决策矩阵只是辅助工具
