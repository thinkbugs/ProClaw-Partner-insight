## 概览

本框架提供系统性的行为跟踪机制，用于识别伴侣格局的变化趋势、验证伪装期、评估改进效果。

**核心原则**:
- 时间序列对比，观察变化而非单点判断
- 记录具体案例，避免记忆偏差
- 设定预警阈值，及时发现问题

**适用场景**:
- 验证皇后型伴侣的真实性（识别伪装期）
- 观察丫鬟型/妃子型伴侣的改进可能性
- 识别关系变化的临界点
- 评估沟通或干预效果

## 跟踪维度

### 维度1:关键事件响应（每月记录）
**记录内容**:
- 伴侣事业上升时的反应（支持/抱怨/控制）
- 伴侣遇到困难时的反应（解决/抱怨/逃避）
- 伴侣做出风险决策时的反应（支持/阻拦/评估）
- 伴侣暂停物质投入时的反应（关心/不满/离开）

**记录模板**:
```
日期: YYYY-MM-DD
事件类型: [事业上升/遇到困难/风险决策/暂停投入]
具体事件: [描述事件详情]
伴侣反应: [具体反应、对话、行为]
你的感受: [主观感受]
得分: [1-10分，参考评估模型]
```

### 维度2:行为模式变化（每季度对比）
**对比维度**:
- 情绪稳定性变化趋势
- 成长支持度变化趋势
- 价值独立性变化趋势
- 自我价值锚点变化趋势

**对比方法**:
- 使用结构化评估模型每季度重新打分
- 对比各维度得分变化
- 识别改进或退化趋势

### 维度3:触发条件识别（实时记录）
**记录内容**:
- 什么情况下她会切换到不同模式
- 什么条件下她会暴露真实格局
- 关系稳定后哪些因素会触发变化

**记录模板**:
```
日期: YYYY-MM-DD
触发条件: [关系稳定/事业上升/遇到困难/物质投入变化]
行为切换: [从什么模式切换到什么模式]
具体表现: [详细描述]
分析: [分析触发机制]
```

### 维度4:边界测试反应（不定期执行）
**测试场景**:
- 加班晚归测试：连续一周加班，观察反应
- 暂停投入测试：暂停物质投入1-2周，观察反应
- 弱点暴露测试：主动暴露失败或脆弱，观察反应
- 风险决策测试：提出一个有风险的决定，观察反应

**记录模板**:
```
测试日期: YYYY-MM-DD
测试类型: [加班/暂停投入/弱点暴露/风险决策]
测试时长: [具体时长]
伴侣反应: [详细记录]
得分变化: [测试前得分 vs 测试后得分]
结论: [结论与建议]
```

## 分析方法

### 方法1:趋势分析
**目的**:识别关系的长期发展方向

**分析方法**:
- 绘制各维度得分的时间序列图
- 识别上升、下降、稳定的维度
- 计算综合得分的变化趋势

**趋势解读**:
- 持续上升（>6个月）→ 真实改进，格局提升
- 持续下降（>3个月）→ 关系恶化，风险增加
- 震荡波动（无明确方向）→ 不稳定，需深入分析原因

### 方法2:临界点识别
**目的**:识别关系变化的转折点

**识别方法**:
- 寻找行为模式突然变化的时间点
- 分析变化触发条件（关系稳定、事业变化、外部事件）
- 判断是阶段调整还是格局暴露

**临界点类型**:
- 伪装期结束:关系稳定6-12个月后
- 格局暴露:遇到重大压力事件后
- 恢复正常:度过危机事件后

### 方法3:改进可能性评估
**目的**:评估丫鬟型/妃子型伴侣的改进空间

**评估维度**:
- 改进意愿:是否愿意沟通、反思、改变
- 改进能力:是否有成长空间和学习能力
- 改进环境:是否支持改变（你的引导、边界设定）
- 格局限制:是否存在深层格局限制（价值观、安全感）

**评估结论**:
- 四项都有 → 有较大改进空间
- 前三项有，格局限制弱 → 中等改进空间
- 前两项有，格局限制强 → 改进空间小
- 仅改进意愿有，其他无 → 基本无改进空间

## 预警机制

### 黄色预警（观察期）
**触发条件**:
- 单维度得分下降2分以上，持续1个月
- 综合得分下降1分以上，持续1个月
- 出现2个以上负面信号（抱怨、控制、阻拦）

**应对措施**:
- 深度沟通，了解原因
- 设定边界，明确期望
- 增加观察频次（每周记录）

### 橙色预警（风险期）
**触发条件**:
- 单维度得分下降3分以上，持续2个月
- 综合得分下降2分以上，持续2个月
- 出现3个以上负面信号，且无改善迹象
- 边界测试得分大幅下降

**应对措施**:
- 评估是否格局暴露（伪装期结束）
- 使用混合特征识别器分析
- 参考决策支持矩阵评估止损时机

### 红色预警（危机期）
**触发条件**:
- 综合得分≤4分
- 情绪稳定性得分≤3分
- 出现危险信号（威胁、控制、暴力倾向）
- 多维度持续退化（>3个月）

**应对措施**:
- 立即使用风险预警系统全面扫描
- 参考决策支持矩阵制定止损策略
- 必要时寻求专业帮助

## 使用流程

### 阶段1:建立基线（第1个月）
- 使用结构化评估模型进行首次评估
- 记录基线得分
- 建立月度记录模板

### 阶段2:常规跟踪（每月）
- 记录本月关键事件
- 更新月度得分
- 识别积极/消极信号
- 制定下月观察重点

### 阶段3:季度对比（每3个月）
- 对比3个月得分变化
- 识别改进/退化维度
- 绘制趋势图
- 评估整体趋势

### 阶段4:临界点分析（随时）
- 识别行为模式变化的时间点
- 分析触发条件
- 判断变化性质（阶段调整 vs 格局暴露）

### 阶段5:预警响应（随时）
- 监控预警条件
- 触发预警后立即分析
- 执行对应应对措施

## 示例

### 示例1:验证皇后型伴侣真实性

**背景**:新伴侣3个月，表现像皇后型，担心是伪装

**跟踪计划**:
- 建立基线（第1个月）:皇后型表现，综合得分8.5分
- 常规跟踪（第2-6个月）:
  - 第2个月:依然良好，得分8.6分
  - 第3个月:关系稳定期，开始测试，得分8.7分
  - 第4个月:压力测试（暂停物质投入），得分下降到7.0分（黄）
  - 第5个月:观察反应，情绪开始不稳定，得分继续下降到6.5分（橙）
  - 第6个月:原形毕露，开始抱怨控制，得分5.0分（红）

**分析结论**:
- 伪装期在关系稳定后结束（第4个月）
- 原本皇后型表现是妃子型伪装
- 压力测试后暴露真实格局
- 建议:转到路径E（危机决策），参考决策支持矩阵

### 示例2:观察丫鬟型伴侣改进

**背景**:在一起2年，丫鬟型，综合得分3.5分，决定尝试沟通改进

**跟踪计划**:
- 建立基线（第1个月）:3.5分
- 深度沟通，设定边界和期望
- 常规跟踪（6个月）:
  - 第1个月:基线3.5分
  - 第2个月:小幅改进3.8分（改进意愿明确）
  - 第3个月:波动3.7分（遇到压力后）
  - 第4个月:继续改进4.0分（黄）
  - 第5个月:稳定在4.2分
  - 第6个月:4.5分，达到中风险区间

**改进可能性评估**:
- 改进意愿:有（明确愿意改变）
- 改进能力:有（学习能力尚可）
- 改进环境:有（你的引导+边界设定）
- 格局限制:中等（安全感问题，价值观基本一致）

**分析结论**:
- 有中等改进空间
- 从高风险（3.5分）改善到中风险（4.5分）
- 建议继续跟踪3个月，观察能否突破到低风险区间（>6分）

### 示例3:识别关系临界点

**背景**:在一起5年，一直稳定，综合得分7.0分，最近3个月突然变化

**时间序列**:
- 前5年:稳定在7.0分（低风险区间）
- 6个月前:你事业大幅上升
- 3个月前:开始频繁抱怨"你心里还有没有这个家"
- 2个月前:阻拦你做风险决策
- 1个月前:综合得分下降到5.0分（中风险）

**临界点分析**:
- 临界点:你事业大幅上升后3个月
- 触发条件:伴侣无法容纳一个不断进化的你
- 变化性质:格局暴露（原本就是丫鬟型，事业上升前不明显）
- 根本原因:伴侣的安全感问题，格局限制

**分析结论**:
- 不是阶段调整，是格局暴露
- 改进可能性评估:改进意愿有，但格局限制强
- 建议:参考决策支持矩阵，评估止损时机

## 注意事项
- 跟踪需要持续3-6个月才能识别真实趋势
- 记录要客观具体，避免记忆偏差
- 不要过度解读短期波动
- 临界点识别需要结合触发条件分析
- 预警机制要严格执行，避免拖延
- 最终决策要结合多个维度综合判断
